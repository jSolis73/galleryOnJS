Галерея реализована на языке javascropt с применением CSS.
Изображения в галерее грузятся со стороны сервера при помощи промисов.
Можно добавлять изображения в галерею при помощи инпута, в который вводится адрес изображения.
Галерея ограничена блоком максимальной шириной 860 пикселей (минимальная - 320 пикселей.).
Ширина изображений в галерее 150 пикселей. Реализован предпросмотр при наведении.
